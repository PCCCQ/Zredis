# Zredis（简约的redis可视化软件）

## 特点

- 管理多个连接
- 增删改查
- 执行redis命令
- 使用element-plus作为UI组件
- 支持Windows，MacOS，Linux
- 体积小，在windows下12mb左右
- 免安装，打开即用
- ......

## 技术栈

golang，wails，vue，element-plus

## 自行编译

先下载wails [The Wails Project | Wails](https://wails.io/zh-Hans/) 和node.js https://nodejs.org/。

下载源码，使用IDE打开。

按照文档 `wails build`进行编译即可。
